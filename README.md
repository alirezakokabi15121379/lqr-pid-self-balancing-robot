# lqr-pid-self-balancing-robot
Designed and simulated a hybrid LQR-PID controller for a two-wheeled self-balancing robot. The hybrid approach dynamically switches between LQR (for small tilt angles) and PID (for larger angles) to achieve robust stabilization up to ±20°, outperforming standalone LQR (limited to ±5°) and PID controllers.
